lolacli 0.2.3

lolacli is a package made for Linux Users (specially newbies). Its written in python and is a CLI which allows you to install/download linux packages/applications/software easily without any excessive commands. The latest version 0.2.3 has the largest app support and we recommend users to use the latest version only. lola uses the click module for designing the base Command Line Interface

Why use lolacli when we have those software managers preinstalled in most of our distributions?

Because graphical software managers are slow! Yes, they are slow. They are definitely slower than the terminal downloads through package managers like APT (Advanced Package Tool), DNF (Dandified YUM}, Flatpak etc. Hence, lolacli package now brings to you an easy, fast and efficient way of installing software! Currently it supports only the distributions which are debian-based and use apt software manager. In future we will try to add more support and make lola use other software managers too.

This project targets Beginner and Advanced Linux Users who don't wanna spend time on adding repositories and installing software with 3 and 4 commands. We will surely improve the product as time passes (if any faults are found)

Example:

Installing Firefox in Ubuntu:

Manual Way: 

$ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys A6DCF7707EBC211F
$ sudo apt-add-repository "deb http://ppa.launchpad.net/ubuntu-mozilla-security/ppa/ubuntu focal main"
$ sudo apt update
$ sudo apt install firefox


The lola Way:



$ lola install firefox
